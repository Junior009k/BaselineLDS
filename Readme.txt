# BASELINE LDS(Login Designa Studios) API

## Descripción

La API de Gestión de Tareas proporciona un conjunto de endpoints para crear, leer, actualizar y eliminar tareas. Esta API está diseñada para ser utilizada en aplicaciones que necesiten gestionar listas de tareas, como aplicaciones de productividad, planificación de proyectos, entre otras.
